The ``7-base_geometry`` module
===============================

Using ``BaseGeometry.area``
----------------------------

First, import ``BaseGeometry``:

    >>> BaseGeometry = __import__('7-base_geometry').BaseGeometry
    >>> bg = BaseGeometry()

``area()`` is not implemented and always raises an ``Exception``:

    >>> bg.area()
    Traceback (most recent call last):
    Exception: area() is not implemented

Using ``BaseGeometry.integer_validator``
------------------------------------------

Valid positive integers pass silently:

    >>> bg.integer_validator("my_int", 12)
    >>> bg.integer_validator("width", 89)
    >>> bg.integer_validator("height", 1)

A non-integer value raises a ``TypeError``:

    >>> bg.integer_validator("name", "John")
    Traceback (most recent call last):
    TypeError: name must be an integer

A float also raises a ``TypeError``, since it's not an ``int``:

    >>> bg.integer_validator("width", 3.14)
    Traceback (most recent call last):
    TypeError: width must be an integer

``None`` raises a ``TypeError`` as well:

    >>> bg.integer_validator("height", None)
    Traceback (most recent call last):
    TypeError: height must be an integer

A value of ``0`` raises a ``ValueError``:

    >>> bg.integer_validator("age", 0)
    Traceback (most recent call last):
    ValueError: age must be greater than 0

A negative value raises a ``ValueError``:

    >>> bg.integer_validator("distance", -4)
    Traceback (most recent call last):
    ValueError: distance must be greater than 0
