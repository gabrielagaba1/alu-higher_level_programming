# python-inheritance

This directory contains exercises on Python inheritance: base and
derived classes, method overriding, multiple inheritance, and
introspection tools such as `dir()`, `type()`, and `isinstance()`.

## Tasks

| File | Description |
| --- | --- |
| `0-lookup.py` | Returns the list of available attributes and methods of an object. |
