# ALU Higher Level Programming

This repository contains projects and exercises completed as part of the
Higher Level Programming curriculum. Each directory corresponds to a
specific project focused on a particular language, concept, or set of
programming techniques.

## Requirements

- Editors allowed: `vi`, `vim`, `emacs`
- All files are interpreted/compiled on Ubuntu 20.04 LTS using `python3`
  (version 3.8.5)
- All files end with a new line
- The first line of every Python file is exactly `#!/usr/bin/python3`
- Code follows the `pycodestyle` style guide (version 2.7.*)
- All files are executable
- Every module, class, and function includes proper documentation

## Projects

### python-classes

Introduces Python classes and object-oriented programming concepts,
starting with simple class definitions and building up to more complex
behavior, attributes, and methods.

| File | Description |
| --- | --- |
| `0-square.py` | Defines an empty `Square` class. |
